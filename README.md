## Pasos para replicar la prueba

### clonar el proyecto

git clone https://github.com/kelvinsinsua/problema3-reactjs.git

### instalar las dependencias

cd problema3-reactjs && yarn install

### iniciar dev server

yarn start

### compilar para producción

yarn build
